Форма создания класса

[class] [newclass] in [(root)]
	generic types [+]
	extends [Class]
	implements [+]
	fields [+]
	constructors [+]
	methods [+]

[class] является элементом выбора типа класса.
[newclass] является элементом ввода строки с валидатором по регулярному выражению и конфликту имен.
[(root)] является элементом выбора пакета с валидатором по конфликту имен классов.
"generic types" является блоком элементов объявления generic-классов.
[Class] является элементом выбора типа с фильтром "не примитивный и не generic" (по умолчанию "com.wizard.js.lang.Class").
"implements" является блоком элементов выбора типа с фильтром "интерфейс" (без повторений).
"fields" является блоком элементов объявления полей с указанием области видимости generic-классов.
"constructors" является блоком элементов объявления конструкторов.
"methods" является блоком элементов объявления методов.

--------

Элемент выбора типа класса

- interface
- abstract class
- class
- final class
- static class

--------

Элемент объявления generic-класса

Всегда раскрывается в форму объявления generic-класса.

Параметры
- область видимости generic-классов

--------

Форма объявления generic-класса

[-name-] extends [Class]
[-name-] является элементом ввода имени generic-класса с валидатором по регулярному выражению и конфликту имен.
[Class] является элементом ввода класса с соответствующей областью видимости.

Параметры
- область видимости generic-классов

--------

Элемент объявления поля

Всегда раскрывается с форму объявления поля.

--------

Форма объявления поля

[private] [-type-] [-name-] = [-value-]
[private] является элементом выбора уровня доступа.
[-type-] является элементом ввода типа.
[-name-] является элементом ввода имени с регулярным выражением и проверкой на конфликт в данном классе.
[-value-] является элементом ввода значения типа [-type-].

Параметры
- область видимости generic-классов

--------

Элемент выбора уровня доступа

- private
- protected
- public
- static

Фильтры:

- protected или public
- public

--------

Элемент выбора модели перегрузки

- abstract
- regular
- final

Фильтры:

- не abstract

--------

Элемент объявления конструктора

Всегда раскрывается в форму объявления конструктора

--------

Форма объявления конструктора

[default]
	arguments [+]
	variables [+]
	actions [+]

[default] является элементом ввода строки с валидатором по регулярному выражению и конфликту имен.
"arguments" является списком элементов объявления аргумента.
"variables" является списком элементов объявления переменной.
"actions" является списком элементов ввода команды с фильтром "не цикл".

--------

Форма ввода конструктора базового класса

super:[-constructor-]()

[-constructor-] является элементом выбора конструктора базового класса.
После выбора конструктора появляются элементы для ввода значений аргументов. Элемент ввода значения аргумента ограничен типом аргумента, указанным в объявлении конструктора.
Если конструктор имеет неопределенное число аргументов, то в конце списка аргументов появляется кнопка [+] для добавления аргумента. Всякий дополнительный аргумент, добавленный кнопкой [+], получит кнопку [-] для его удаления.

--------

Элемент объявления метода

- define method
- override method

"define method" раскрывается в форму объявления метода.
"override method" раскрывается в форму перегрузки метода.

--------

Форма объявления метода

[private] [regular] [-type-] [-name-]
	arguments [+]
	variables [+]
	actions [+]

[private] является элементом выбора уровня доступа.
[regular] является элементом выбора модели перегрузки.
[-type-] является элементом ввода типа.
[-name-] является элементом ввода имени с регулярным выражением и проверкой на конфликт в данном классе.
"arguments" является списком элементов объявления аргумента.
"variables" является списком элементов объявления переменной.
"actions" является списком элементов ввода команды с фильтрами "не конструктор", "не цикл".

--------

Форма перегрузки метода

override [-method-] [protected] [regular] [-type-]
	arguments
	variables [+]
	actions [+]

[-method-] является элементом выбора метода базового класса с фильтрами "только публичные и защищенные" и "не final".
[protected] является элементом выбора уровня доступа с фильтром не ниже уровня доступа метода базового класса.
[regular] является элементом выбора модели перегрузки с фильтром "не abstract".
[-type-] является неактивным элементом ввода типа возвращаемого значения.
"arguments" является нередактируемым блоком неактивных элементов объявления аргументов.
"variables" является блоком элементов объявления переменных.
"actions" является блоком элементов ввода команд с фильтрами "не конструктор", "не цикл".

--------

Элемент объявления аргумента

Всегда раскрывается в форму объявления аргумента.

--------

Форма объявления аргумента

[-type-] [-name-]

[-type-] [-name-] = [-value-]
[-type-] является элементом ввода типа.
[-name-] является элементом ввода имени с регулярным выражением и проверкой на конфликт в текущей области объявления.
